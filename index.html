Módulo 1.1: Variables y Tipos de Datos

Explicación

Una variable es como una caja con una etiqueta donde puedes guardar información (valores). En Python, no necesitas declarar el tipo de dato de una variable antes de usarla (esto se llama tipado dinámico). Python lo infiere automáticamente. Los tipos de datos básicos son:

  • int: Números enteros (ej: 10, -3, 0).
  • float: Números con decimales (ej: 3.14, -0.5, 2.0).
  • str: Cadenas de texto (ej: "Hola", 'Python123').
  • bool: Valores booleanos, que solo pueden ser True (verdadero) o False (falso).

Ejemplos de Código

# Variables y asignación
nombre_del_curso = "Python Fundamentos"  # str
numero_estudiantes = 25                 # int
precio_curso = 49.99                    # float
es_online = True                        # bool

# Imprimir variables
print(nombre_del_curso)
print(numero_estudiantes)
print(precio_curso)
print(es_online)

# Puedes ver el tipo de una variable con type()
print(type(nombre_del_curso)) # Salida: <class 'str'>

Ejercicios para Practicar

  1. Crea variables para almacenar tu nombre, tu edad, tu altura en metros (ej: 1.75) y si te gusta el café (True/False). Luego, imprime el valor de cada variable.
  2. Pide al usuario que ingrese su ciudad natal usando la función input(), almacena la respuesta en una variable e imprímela.
  3. Declara dos variables numéricas, a = 10 y b = 3. Calcula e imprime su suma, resta, multiplicación y división.

Ejercicios de Lógica de Programación

  1. Piensa cómo representarías la información de un libro: título, autor, número de páginas, y si está disponible en formato digital. ¿Qué tipos de datos usarías para cada pieza de información?
  2. Si tienes el precio de un producto y la cantidad que un cliente quiere comprar, ¿cómo calcularías el costo total? ¿Qué variables necesitarías?

Trucos y Consejos

  • Usa nombres de variables descriptivos y en minúsculas, separando palabras con guiones bajos (ej: mi_variable_descriptiva). Esto se llama "snake_case".
  • Los comentarios en Python empiezan con #. Úsalos para explicar tu código: # Esto es un comentario.
  • Puedes convertir entre algunos tipos de datos, ej: int("10") convierte la cadena "10" a un entero. str(25) convierte el entero 25 a una cadena.

Módulo 1.2: Operadores

Explicación

Los operadores son símbolos que realizan operaciones sobre variables y valores.

  • Aritméticos: + (suma), - (resta), * (multiplicación), / (división), // (división entera), % (módulo), ** (exponenciación).
  • De Asignación: =, +=, -=, *=, /=.
  • De Comparación: ==, !=, <, >, <=, >=. Devuelven True o False.
  • Lógicos: and, or, not. Se usan con valores booleanos.

Ejemplos de Código

# Operadores Aritméticos
a = 15
b = 4
print(f"Suma: {a + b}")           # 19
print(f"División entera: {a // b}") # 3
print(f"Módulo: {a % b}")         # 3
print(f"Potencia: {2 ** 3}")      # 8 (2 elevado a 3)

# Operadores de Asignación
contador = 0
contador += 5  # contador ahora es 5
print(f"Contador: {contador}")

# Operadores de Comparación
edad_juan = 20
edad_minima = 18
es_mayor = edad_juan >= edad_minima
print(f"¿Juan es mayor de edad? {es_mayor}") # True

# Operadores Lógicos
tiene_entrada = True
es_vip = False
puede_pasar = tiene_entrada and not es_vip
print(f"¿Puede pasar? {puede_pasar}")

Ejercicios para Practicar

  1. Pide al usuario dos números. Muestra la suma, resta, producto, división, división entera y el módulo de ambos.
  2. Pide al usuario su año de nacimiento. Calcula su edad aproximada e imprímela.
  3. Declara una variable temperatura = 25. Escribe expresiones que evalúen a True o False para:
    • ¿La temperatura es mayor a 30?
    • ¿La temperatura es menor o igual a 25?
    • ¿La temperatura no es igual a 20?

Ejercicios de Lógica de Programación

  1. Para que un usuario pueda acceder a una sección especial de una web, necesita ser mayor de edad (>= 18) Y tener una suscripción activa (una variable booleana). Escribe la lógica (la expresión booleana) que determine si puede acceder.
  2. Un producto tiene un descuento del 10% si el precio original es mayor a 50€. ¿Cómo expresarías la condición para aplicar el descuento?

Trucos y Consejos

  • Usa paréntesis () para agrupar operaciones y controlar el orden de evaluación si tienes dudas sobre la precedencia de operadores.
  • Las f-strings (como f"Hola, {nombre}") son una forma muy cómoda de incluir variables dentro de cadenas de texto.

Módulo 2.1: `if / elif / else` (Condicionales)

Explicación

Permiten ejecutar diferentes bloques de código según si una condición es verdadera o falsa.

  • if: Ejecuta el bloque si la condición es True.
  • elif (else if): Se evalúa si el if anterior (o elif anterior) fue False. Puedes tener varios elif.
  • else: Ejecuta el bloque si todas las condiciones anteriores (if y elif) fueron False. Es opcional.

Ejemplos de Código

nota = float(input("Ingresa tu nota (0-10): "))

if nota >= 9:
    print("Sobresaliente")
elif nota >= 7:
    print("Notable")
elif nota >= 5:
    print("Aprobado")
else:
    print("Suspenso")

# Ejemplo simple
numero = -5
if numero > 0:
    print("El número es positivo.")
else:
    print("El número es cero o negativo.")

Ejercicios para Practicar

  1. Pide al usuario un número. Indica si el número es par o impar. (Pista: usa el operador módulo %).
  2. Pide al usuario su edad. Si es menor de 13 años, muestra "Eres un niño". Si tiene entre 13 y 17 (inclusive), muestra "Eres un adolescente". Si tiene 18 o más, muestra "Eres un adulto".
  3. Pide dos números al usuario. Indica cuál es el mayor o si son iguales.

Ejercicios de Lógica de Programación

  1. Un semáforo tiene tres colores: "rojo", "amarillo", "verde". Diseña la lógica if/elif/else que indique la acción a tomar para cada color (ej: "rojo" -> "Detenerse").
  2. Para obtener un descuento en una tienda, un cliente debe ser miembro Y su compra debe superar los 100€, O BIEN, si no es miembro, su compra debe superar los 200€. ¿Cómo estructurarías esta lógica condicional?

Trucos y Consejos

  • ¡La indentación (los espacios al inicio de la línea) es CRUCIAL en Python! Define qué código pertenece a cada bloque (if, elif, else). Usa 4 espacios por nivel de indentación.
  • Puedes anidar condicionales (un if dentro de otro if), pero intenta no hacerlo demasiado complejo.

Módulo 2.2: `for` loops (Bucles Iterativos)

Explicación

Un bucle for se utiliza para iterar (repetir un bloque de código) sobre los elementos de una secuencia (como una lista, una tupla, una cadena de texto) o sobre un rango de números.

Ejemplos de Código

# Iterar sobre una lista
frutas = ["manzana", "banana", "cereza"]
for fruta in frutas:
    print(f"Me gusta la {fruta}")

# Iterar sobre una cadena de texto
palabra = "Python"
for letra in palabra:
    print(letra)

# Iterar un número específico de veces usando range()
# range(5) genera números del 0 al 4
for i in range(5):
    print(f"Número: {i}")

# range(inicio, fin, paso)
# Imprimir números pares del 2 al 10
for num in range(2, 11, 2): # 2, 4, 6, 8, 10
    print(num)

Ejercicios para Practicar

  1. Imprime los números del 1 al 100.
  2. Pide al usuario un número n. Imprime la tabla de multiplicar de ese número del 1 al 10 (ej: si n=5, imprime "5x1=5", "5x2=10", ..., "5x10=50").
  3. Dada una lista de nombres: nombres = ["Ana", "Luis", "Eva", "Juan"], crea un nuevo mensaje de saludo personalizado para cada nombre e imprímelo.

Ejercicios de Lógica de Programación

  1. Tienes una lista de precios de productos. ¿Cómo calcularías el precio total de todos los productos usando un bucle for y una variable acumuladora?
  2. ¿Cómo encontrarías la palabra más larga en una lista de palabras usando un bucle for? (Pista: necesitarás una variable para guardar la palabra más larga encontrada hasta el momento y otra para su longitud).

Trucos y Consejos

  • La variable que usas en el for (ej: fruta en for fruta in frutas:) toma el valor de cada elemento de la secuencia en cada iteración.
  • range(N) genera números de 0 a N-1.
  • Puedes usar break para salir de un bucle prematuramente y continue para saltar a la siguiente iteración del bucle.

Módulo 2.3: `while` loops (Bucles Condicionales)

Explicación

Un bucle while ejecuta un bloque de código repetidamente mientras una condición especificada sea True.

Ejemplos de Código

contador = 0
while contador < 5:
    print(f"El contador es {contador}")
    contador += 1  # ¡Importante! Actualizar la variable de control
print("Fin del bucle while")

# Bucle que se ejecuta hasta que el usuario ingrese 'salir'
entrada = ""
while entrada.lower() != "salir":
    entrada = input("Escribe algo (o 'salir' para terminar): ")
    print(f"Escribiste: {entrada}")

Ejercicios para Practicar

  1. Crea un programa que pida números al usuario y los sume, hasta que el usuario ingrese un 0. Al final, muestra la suma total.
  2. Simula una cuenta atrás desde un número ingresado por el usuario hasta 1, imprimiendo cada número.
  3. Pide al usuario que adivine una contraseña secreta. El bucle debe continuar hasta que la adivine. (Para empezar, puedes poner la contraseña directamente en el código).

Ejercicios de Lógica de Programación

  1. Quieres simular un juego donde un personaje tiene 100 puntos de vida y recibe ataques que le restan una cantidad aleatoria de vida (entre 5 y 15 puntos) en cada turno. El juego termina cuando la vida del personaje es 0 o menos. ¿Cómo usarías un bucle while para esto?
  2. ¿Cómo validarías que la entrada del usuario sea un número positivo usando un bucle while? (El bucle debería repetirse si ingresa un número negativo o cero).

Trucos y Consejos

  • ¡Cuidado con los bucles infinitos! Asegúrate de que la condición del while eventualmente se vuelva False.
  • Un while True: con un break dentro es una forma común de crear bucles que terminan bajo ciertas condiciones internas.

Módulo 3.1: Listas (Lists)

Explicación

Las listas son colecciones ordenadas y mutables de elementos. Se acceden por índice (empezando en 0).

  • Creación: mi_lista = [1, "hola", 3.0]
  • Acceso: mi_lista[0], mi_lista[-1] (último).
  • Slicing: mi_lista[1:3].
  • Métodos: append(), insert(), remove(), pop(), sort(), len().

Ejemplos de Código

numeros = [10, 5, 25, 15]
print(f"Lista original: {numeros}")

numeros.append(30)
print(f"Después de append: {numeros}")

numeros.sort()
print(f"Lista ordenada: {numeros}")

print(f"Elemento en índice 1: {numeros[1]}")
print(f"Longitud de la lista: {len(numeros)}")

Ejercicios para Practicar

  1. Crea una lista con los nombres de tus 3 ciudades favoritas.
    • Añade una ciudad más al final.
    • Inserta una ciudad al principio.
    • Elimina la segunda ciudad.
    • Imprime la lista final y su longitud.
  2. Pide al usuario 5 números, guárdalos en una lista. Calcula e imprime la suma y el promedio.

Ejercicios de Lógica de Programación

  1. Tienes una lista de notas. Cuenta cuántos aprobaron (>= 5).
  2. Crea una nueva lista solo con los números pares de una lista original.

Trucos y Consejos

  • Las listas pueden tener elementos de diferentes tipos.
  • Usa in para verificar si un elemento está en una lista.
  • Copia listas con lista_copia = lista_original[:] o list(lista_original).

Módulo 3.2: Tuplas (Tuples)

Explicación

Las tuplas son colecciones ordenadas e inmutables. No pueden cambiarse después de su creación.

  • Creación: mi_tupla = (1, "hola", 3.0) o 1, "hola", 3.0. Un solo elemento: (5,).
  • Acceso y Slicing: Igual que las listas.

Ejemplos de Código

punto = (10, 20, 5) # Coordenadas x, y, z
print(f"Coordenada X: {punto[0]}")

# Esto daría un error:
# punto[0] = 15

# Desempaquetado de tuplas
nombre, edad, ciudad = ("Ana", 30, "Madrid")
print(f"{nombre} tiene {edad} años y vive en {ciudad}.")

Ejercicios para Practicar

  1. Crea una tupla con los meses del año. Imprime el primer y último mes.
  2. Crea una tupla para un estudiante: (id, nombre, curso).
  3. Intenta añadir un elemento a una tupla y observa el error.

Ejercicios de Lógica de Programación

  1. Si una función devuelve varios valores, ¿es una tupla una buena opción? ¿Por qué?
  2. ¿Cómo iterarías sobre los elementos de una tupla?

Trucos y Consejos

  • Las tuplas pueden ser más eficientes si los datos no cambian.
  • Útiles como claves en diccionarios por ser inmutables.

Módulo 3.3: Diccionarios (Dictionaries)

Explicación

Colecciones no ordenadas (orden de inserción en Python 3.7+) y mutables de pares clave-valor. Claves únicas e inmutables.

  • Creación: {"clave1": "valor1", "edad": 25}
  • Acceso: mi_diccionario["clave"].
  • Añadir/Modificar: mi_diccionario["nueva"] = "valor".
  • Métodos: get(), keys(), values(), items().

Ejemplos de Código

capitales = {"España": "Madrid", "Francia": "París"}
print(capitales["Francia"])

capitales["Alemania"] = "Berlín"
print(capitales.get("Portugal", "No encontrada"))

for pais, capital in capitales.items():
    print(f"Capital de {pais}: {capital}.")

Ejercicios para Practicar

  1. Crea un diccionario de precios de productos.
    • Imprime el precio de un producto.
    • Añade un nuevo producto.
    • Modifica un precio.
    • Muestra todos los productos (claves).
  2. Crea un diccionario para un estudiante: "nombre", "asignaturas" (lista), "notas" (lista).

Ejercicios de Lógica de Programación

  1. Cuenta la frecuencia de cada letra en una frase usando un diccionario.
  2. Calcula el costo total de una lista de compra usando un diccionario de precios. Maneja productos no encontrados.

Trucos y Consejos

  • Las claves son sensibles a mayúsculas/minúsculas.
  • Usa in para verificar si una clave existe.
  • Elimina con del o pop().

Módulo 3.4: Conjuntos (Sets)

Explicación

Colecciones no ordenadas y mutables de elementos únicos. No permiten duplicados.

  • Creación: {1, 2, 3, 2} (resulta en {1, 2, 3}). Vacío: set().
  • Métodos: add(), remove(), discard(), union(), intersection(), difference().

Ejemplos de Código

numeros_loteria = {7, 15, 23, 30, 15, 42}
print(f"Números: {numeros_loteria}")

numeros_loteria.add(10)
print(f"Añadido 10: {numeros_loteria}")

mis_numeros = {10, 20, 30, 40}
comunes = numeros_loteria.intersection(mis_numeros)
print(f"Comunes: {comunes}")

todos = numeros_loteria.union(mis_numeros)
print(f"Unión: {todos}")

Ejercicios para Practicar

  1. Crea dos conjuntos de nombres de amigos.
    • Encuentra amigos comunes (intersección).
    • Encuentra todos los amigos únicos (unión).
    • Encuentra amigos en el primer conjunto pero no en el segundo (diferencia).
  2. Convierte una lista con duplicados a un conjunto y luego de nuevo a lista para eliminar duplicados.

Ejercicios de Lógica de Programación

  1. Obtén una lista de emails activos (registrados pero no cancelados) usando dos listas de emails.
  2. Verifica si todos los elementos de una lista pequeña están en un conjunto más grande.

Trucos y Consejos

  • Eficientes para verificar existencia de un elemento (in).
  • No se accede por índice (no ordenados).

Módulo 4: Funciones

Explicación

Bloques de código reutilizables que realizan una tarea específica. Ayudan a organizar y evitar repetición.

  • Definición: def nombre_funcion(parametros):. Código indentado.
  • Parámetros: Variables que recibe la función.
  • Retorno: Usa return valor. Devuelve None si no hay return.
  • Docstrings: """Explica qué hace la función.""" al inicio.

Ejemplos de Código

def saludar(nombre):
    """Devuelve un saludo personalizado."""
    mensaje = f"¡Hola, {nombre}! Bienvenido/a."
    return mensaje

print(saludar("Ana"))

def sumar(a, b):
    return a + b
print(f"5 + 3 = {sumar(5, 3)}")

def imprimir_cuadrado(numero):
    """Imprime el cuadrado. No devuelve nada explícitamente."""
    print(f"El cuadrado de {numero} es {numero ** 2}")

valor_retornado = imprimir_cuadrado(5)
print(f"Valor retornado: {valor_retornado}") # None

Ejercicios para Practicar

  1. Función calcular_area_rectangulo(base, altura) que devuelva el área.
  2. Función es_par(numero) que devuelva True si es par, False si no.
  3. Función convertir_a_mayusculas(cadena) que devuelva la cadena en mayúsculas (usa .upper()).
  4. Función que reciba una lista de números y devuelva su suma.

Ejercicios de Lógica de Programación

  1. Función obtener_calificacion(nota) que devuelva "Suspenso", "Aprobado", etc.
  2. Función para determinar si una palabra es un palíndromo (devuelve True/False).

Trucos y Consejos

  • Nombres descriptivos (verbos).
  • Funciones deben hacer una cosa bien.
  • Parámetros con valores por defecto: def func(nombre, saludo="Hola"):.
  • Variables locales solo existen dentro de la función.

Módulo 5: Manejo de Errores

Explicación

Los errores (excepciones) pueden ocurrir. Manejarlos evita que el programa se detenga.

  • try: Código que podría causar error.
  • except TipoDeError:: Se ejecuta si ocurre ese error en try.
  • except Exception as e:: Captura cualquier error.
  • else: (Opcional) Si no hubo excepción en try.
  • finally: (Opcional) Se ejecuta siempre.

Errores Comunes: ValueError, TypeError, IndexError, KeyError, ZeroDivisionError, FileNotFoundError.

Ejemplos de Código

try:
    dividendo = float(input("Dividendo: "))
    divisor = float(input("Divisor: "))
    resultado = dividendo / divisor
    print(f"Resultado: {resultado}")
except ValueError:
    print("Error: Ingresa números válidos.")
except ZeroDivisionError:
    print("Error: No se puede dividir por cero.")
except Exception as e:
    print(f"Error inesperado: {e}")
else:
    print("División exitosa.")
finally:
    print("--- Fin del intento ---")

mi_lista = [10, 20]
try:
    indice = int(input("Índice: "))
    print(mi_lista[indice])
except IndexError:
    print("Error: Índice fuera de rango.")
except ValueError:
    print("Error: Ingresa un número entero.")

Ejercicios para Practicar

  1. Modifica el ejercicio de división para manejar ValueError y ZeroDivisionError.
  2. Pide un mes y muestra sus días (usa un diccionario). Maneja KeyError.
  3. Función que intente convertir cadena a entero. Devuelve entero o None si falla.

Ejercicios de Lógica de Programación

  1. Carga una configuración de forma segura usando try-except, con valores por defecto si falla.
  2. Conexión a servicio externo: usa try-except y un bucle para reintentar antes de rendirse.

Trucos y Consejos

  • Sé específico con el tipo de error en except.
  • Evita except Exception: genérico si es posible.
  • Registra errores o muéstralos amigablemente.

Módulo 6: Entrada/Salida Básica (Archivos)

Explicación

Aprenderás a leer y escribir datos en archivos de texto.

  • open(nombre_archivo, modo): Abre un archivo.
    • Modos: 'r' (leer), 'w' (escribir, sobrescribe), 'a' (añadir), 'r+' (leer/escribir).
  • Lectura: archivo.read(), archivo.readline(), archivo.readlines().
  • Escritura: archivo.write(cadena).
  • archivo.close(): ¡Muy importante!
  • with open(...) as archivo:: Recomendado, cierra automáticamente.

Ejemplos de Código

# Escribir (sobrescribe)
try:
    with open("mi_archivo.txt", "w") as f:
        f.write("Hola, Python!\n")
        f.write("Segunda línea.\n")
    print("Archivo escrito.")
except IOError:
    print("Error al escribir.")

# Leer completo
try:
    with open("mi_archivo.txt", "r") as f:
        contenido = f.read()
        print("\nContenido:\n" + contenido)
except FileNotFoundError:
    print("Error: Archivo no encontrado.")

# Leer línea por línea
try:
    with open("mi_archivo.txt", "r") as f:
        print("\nLínea por línea:")
        for linea in f:
            print(linea.strip()) # .strip() quita espacios/saltos
except FileNotFoundError:
    print("Error: Archivo no encontrado.")

# Añadir contenido
try:
    with open("mi_archivo.txt", "a") as f:
        f.write("Línea añadida.\n")
    print("\nContenido añadido.")
    with open("mi_archivo.txt", "r") as f_ver:
        print("Verificando:\n" + f_ver.read())
except IOError:
    print("Error al añadir.")

Ejercicios para Practicar

  1. Pide nombre y edad, guárdalos en usuario.txt (cada dato en nueva línea).
  2. Lee usuario.txt y muestra la información.
  3. Crea lista_compras.txt. Lee y muestra artículos numerados.
  4. Modifica el anterior para permitir añadir nuevos artículos a lista_compras.txt.

Ejercicios de Lógica de Programación

  1. Archivo con un número por línea. Lee, suma y muestra el total (maneja ValueError).
  2. Copia el contenido de origen.txt a destino.txt línea por línea.

Trucos y Consejos

  • Usa siempre with open(...).
  • .strip() es útil para limpiar líneas leídas.
  • Maneja FileNotFoundError.

Proyectos Prácticos Integradores (Etapa 1)

Ahora es el momento de combinar todo lo que has aprendido. Intenta realizar estos proyectos:

1. Calculadora Mejorada

  • Crea funciones para cada operación (suma, resta, multiplicación, división).
  • Pide al usuario dos números y la operación que desea realizar.
  • Maneja errores si el usuario ingresa texto en lugar de números o intenta dividir por cero.
  • Permite al usuario realizar múltiples cálculos hasta que decida salir.

2. Juego de Adivinanza de Números Avanzado

  • El programa genera un número aleatorio entre 1 y 100 (investiga el módulo random, random.randint(a, b)).
  • El usuario intenta adivinarlo.
  • Después de cada intento, el programa dice si el número ingresado es mayor o menor que el secreto.
  • Cuenta el número de intentos.
  • Opcional: Guarda el nombre del jugador y su mejor puntuación (menos intentos) en un archivo.

3. Gestor Simple de Lista de Tareas

  • Permite al usuario: Añadir tarea, Ver tareas (numeradas), Marcar como completada, Eliminar tarea, Salir.
  • Usa funciones para cada opción del menú.
  • Guarda la lista de tareas en un archivo (tareas.txt) y cárgala al inicio.

Sobre el Mejor IDE para Python

Como se menciona en el "Tutorial de Python para Principiantes", la elección del IDE depende de tus preferencias. Para principiantes, algunas excelentes opciones son:

  • IDLE: Viene con Python, es simple y bueno para empezar.
  • Visual Studio Code (VS Code): Muy popular, ligero, versátil y con muchas extensiones para Python. Altamente recomendado.
  • PyCharm (Community Edition): Un IDE potente y específico para Python, con muchas herramientas útiles para escribir código limpio.
  • Jupyter Notebook: Excelente para aprendizaje interactivo, ciencia de datos y experimentar con código en "celdas".

Te sugiero probar VS Code o PyCharm Community Edition una vez que te sientas cómodo con los conceptos básicos. ¡Ambos son excelentes para desarrollar proyectos más grandes!
